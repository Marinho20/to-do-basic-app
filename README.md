Python To do App

A command-line task manager built in Python. Add, view, remove, and mark tasks as complete. Completed tasks can also be deleted in bulk. Tasks are saved to a file, so your list persists between sessions. Can be packaged as a standalone Windows .exe.

Features:
->Add tasks with automatic ID assignment
->View tasks with status (Pending / Complete)
->Mark tasks as complete
->Remove individual tasks
->Remove all completed tasks
->Persistent storage using a text file
->Modular design with separate logic and menu
-> Windows .exec file build via PyInstaller


Usage

1.Run main.py
2.Follow the menu:
3.Manage Tasks → add, remove, mark complete
4.View Tasks → list all tasks
5.Remove Completed Tasks → clear all completed tasks
6.Save & Exit → save changes and close
7.Tasks are automatically saved to tasks.txt.


Learning Outcomes

->Python file I/O and data persistence
->List and tuple manipulation
->Modular project structure (logic vs UI)
->Basic software design principles
->Packaging Python apps as standalone executables
